# Dependencies

+ [node](http://nodejs.org/)

+ [npm](https://npmjs.org/)
```shell
curl https://npmjs.org/install.sh | sudo sh
```

# Build

+ Install dev dependencies
```shell
npm install
```

+ Lint less
```shell
./node_modules/recess/bin/recess less/style.less
```

+ Run build
```shell
grunt build
```
